Testing II - Testing in React

How Do We Test React Code?

  • We are using the React Testing Library (link)
  • We use it in conjunction with Jest

RTL Cheatsheet

  1. Create Test File Next to Component Component.test.jsx.
  2. Import React, RTL, & Component To Test
  3. Create Describe & It
describe("Component", () => {
  it('outputs "hello world"', () => {
    // assertions go here
  });
});

Tips From Gabe

  1. We need stuff from the library but the wrapper as well
  2. remember about cleaning up after each test
  3. prefer regular expressions to raw text
  4. wrapper exposes different methods that look alike!!!!

  • query will return null if no match
