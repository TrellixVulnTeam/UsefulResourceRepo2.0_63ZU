Interview PromptExpectations
3
Can you show me where you used the Router, Route, and Link components in your code?Student has at least two routes in their app - home and form
4
Can you show me how you controlled inputs in your form?Student controls multiple types of inputs, AND all inputs have a value/checked prop driven by component state, AND change handlers are attached to all inputs, AND student has a functional submit handler that submits the form
5
Can you show me how you validate user input?Student validates user input in a sensible manner, AND the form cannot be submitted with validation errors, AND the user can see validation errors in the UI
6
Can you show me where in your code you make POST requests?Student's POST request succeeds and resolved promise is handled (if only to log the data).Rejected promise is handled (if only to log the error)
7
Can you show me the tests you wrote and explain what they do?Student uses Cypress to test that it's possible to type values inside one of the inputs, that it's possible to select multiple toppings, and that it's possible to submit the form
