Interview PromptExpectations
3
How did you handle password data?Student uses environment variables to make the number of rounds used to hash the password dynamic, so that the number can be increased on production servers.
4
Can you explain your authentication flow?Student uses environment variables to store cookies or JWT secrets, OR if using Sessions, student stores the sessions on the database instead of in memory, OR student implements authentication twice - one using sessions and one using JWT.
5
Can you show me your unit tests and describe how they work?
Student writes at least 8 unit tests for non router and non database code like validation, mapping, business logic, etc.
6
Can you show me your endpoint tests and describe how they work?Student uses a separate testing database for running the tests, and writes tests for middleware code.
